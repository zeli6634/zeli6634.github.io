About this Project

 From the magnifier, I learned how to use the magnifier on Mac. I tried the zoom for multiple times. I am a high myopia person, so I tried the zoom for 200%. Although I still cannot see things very clear without, but it definitely clearer than normal mode. The biggest challenge I faced is I never use the magnifier before so I look through the tutorial and make sure I can do any step correct. Also remember everything is hard, so I tried my best to remember the command in order to use it in the future.

For the low vision mode, I made the modification for the font size, color of text, the size of the input box and button, and the background color. The style is different than the normal mode, but most of the layout are the same.
